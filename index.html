Comment ça marche

Les bannières actives sont automatiquement insérées dans les signatures email de vos collaborateurs. Seule la bannière de plus haute priorité est affichée. Les clics sont comptabilisés via une redirection de tracking. Vous pouvez planifier des campagnes en définissant des dates de début et de fin.

Paramètres

Informations entreprise
Microsoft Outlook 365
Mode démonstration

Pour activer la synchronisation automatique avec Outlook 365, configurez les variables suivantes dans le fichier .env à la racine du projet :

MS_CLIENT_ID=votre-client-id
MS_CLIENT_SECRET=votre-client-secret
MS_TENANT_ID=votre-tenant-id

Créez une application dans Azure AD avec les permissions Mail.ReadWrite et User.Read.All.
